CryptoGuy
=========

CryptoGuy is supposed to be a tool to be use to figure out which algo is being used for any random given string.

Usage
-----

This project uses a standard CMake based build.
Compile using the following commands:

```
mkdir build
cd build
cmake ../src
make
```
